========================================================================
                     律師國考刷題系統 - 建置與維護工具說明書
========================================================================

本資料夾（build_tools/）存放了用於解析考題、編譯詳解、生成複習卡與聽讀段落，
以及預備部署的輔助工具。平時網站運作不需要執行這些檔案，僅在您「更新詳解」
或「更新考題」時需要使用。

根據使用情境，這些工具的用途與分類如下：

------------------------------------------------------------------------
🌟 類別一：核心維護工具（修改詳解、準備部署時「必用」）
------------------------------------------------------------------------

1. md_to_js.py
   - 功用：將「詳解/*.md」中的中文解析內容，編譯並輸出至「網站/js/explanations_data.js」資料庫。
   - 使用時機：當您在 Markdown 中新增或修改了解析內容時執行。
   - 執行指令：python build_tools/md_to_js.py

2. generate_flashcards.py
   - 功用：重新生成前端「重點整理（概念複習）」所使用的卡片資料（輸出至 js/generated_flashcards.js）。
   - 使用時機：更新詳解後，或是修改了複習卡大綱時執行。
   - 執行指令：python build_tools/generate_flashcards.py

3. generate_audio_scripts.py
   - 功用：重新生成前端「聽讀功能」所使用的段落朗讀與翻譯文本（輸出至 js/audio_scripts.js）。
   - 使用時機：更新詳解後執行，以同步朗讀段落。
   - 執行指令：python build_tools/generate_audio_scripts.py

4. deploy_and_compress.py
   - 功用：一鍵將「網站/」下所有最新的網頁程式碼與資料庫同步複製到「部署/」資料夾中，
           同時對四大 JavaScript 主檔案（explanations_data.js 等）進行最高規格的 gzip 預壓縮（產生 .js.gz 檔）。
   - 使用時機：當您完成本機測試，準備將網站部署上線（如 Zeabur）前執行。
   - 執行指令：python build_tools/deploy_and_compress.py


------------------------------------------------------------------------
🔍 類別二：診斷與排錯工具（檢查格式錯誤、校正標籤用「建議保留」）
------------------------------------------------------------------------

5. diagnose_explanations.py
   - 功用：全面掃描並診斷目前「題庫」與「詳解」的對齊狀況，回報是否有漏寫詳解的題目（Coverage 覆蓋率報告）。
   - 使用時機：在準備部署上線前，用來確認有沒有漏掉任何一題詳解。
   - 執行指令：python build_tools/diagnose_explanations.py

6. sort_md_files.py
   - 功用：物理排序「詳解/*.md」內的所有題目區塊，自動調整為：年份最新到舊（DESC）、題號由小到大（ASC）。
   - 使用時機：當您在 Markdown 中任意貼上新題目後，執行此檔案可自動把內容排整齊。
   - 執行指令：python build_tools/sort_md_files.py

7. auto_align_headers.py / check_alignment.py
   - 功用：根據題庫內容，自動比對並修正 Markdown 檔案中打錯或寫錯的題目標題標籤（例如不小心打錯題號或年份）。
   - 使用時機：編譯詳解出現錯配警告時執行。
   - 執行指令：python build_tools/auto_align_headers.py


------------------------------------------------------------------------
📦 類別三：題庫導入工具（平時「不需要」執行）
------------------------------------------------------------------------

8. build.py
   - 功用：讀取「一試考古題/」目錄下的原始考題文字檔，解析並生成前端的主題庫「js/questions_data.js」。
   - 使用時機：目前 107~114 年的考題已經完全建置固定。除非未來有新一年的考試（如 115 年），
           您放入新的原始試卷文字檔時，才需要執行此腳本來導入新題庫。
   - 執行指令：python build_tools/build.py

========================================================================
💡 標準更新工作流：
   修改 詳解.md -> 執行編譯與生成（1、2、3）-> 執行檢查（5）-> 執行部署打包（4）
========================================================================
